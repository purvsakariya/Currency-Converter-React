# 💱 React Currency Converter

A modern **Currency Converter Web App** built using **React + Vite + Tailwind CSS**.
This application allows users to convert currencies in real-time using live exchange rate data.

---

## 🚀 Features

* Convert between multiple world currencies
* Real-time exchange rate fetching
* Swap currencies instantly
* Clean and responsive UI
* Built with modern React Hooks
* Fast performance using Vite

---

## 🛠️ Tech Stack

* **React.js**
* **Vite**
* **Tailwind CSS**
* **JavaScript (ES6+)**
* **Custom React Hooks**

---

## 📸 Preview

Currency converter interface with:

* Currency selection dropdown
* Amount input
* Swap button
* Real-time conversion

---

## 🧠 How It Works

The application uses a **custom React hook** (`useCurrencyInfo`) to fetch currency exchange rates from an API.
The user selects the source currency and the target currency, enters an amount, and the app calculates the converted value instantly.
